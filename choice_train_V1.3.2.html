    Documentation and terminology
    EEG research mode and active control condition

    EEG research mode adds precise event markers, short fixed blocks, automatic pauses, and ERP-friendly exports for later EEG alignment.

    Block size determines how many scored trials run before an automatic pause appears.

    Active control keeps the same art, reward, and session structure but reduces the core attentional demand so later publication work has a matched comparison condition.

    Marker exports are intended for research processing rather than for teachers to interpret directly.

    How the main session settings work

    Auto advance moves to the next question after feedback. If it is off, the learner or practitioner uses the on-screen next controls.

    Reward screen shows the success screen between questions. With auto advance on, it becomes a short pause before the next item.

    Single-target first stage keeps the early developmental levels focused on one target before the later reasoning tasks appear.

    Auto level path follows the staged ChoiceTrain sequence. If it is off, the selected session task stays fixed instead.

    Question window is the maximum time allowed for a response before the app records a no-response outcome.

    What each auto-path level is doing

    Level 1 builds screen-touch confidence. Any touch in the active panel counts.

    Level 2 requires touching the pictured target itself.

    Level 3 introduces a moving target.

    Level 4 asks the learner to find a named object from a small set.

    Level 5 increases discrimination demand between similar-looking items.

    Level 6 uses category reasoning.

    Levels 7 to 10 move into the CPAT-style attention suite: sustained, selective-spatial, orienting, and executive attention.

    CPAT and attention terms

    Sustained attention means staying ready over time and responding only when the target appears.

    Selective-spatial attention means finding the correct target among distractors while ignoring similar but wrong options.

    Orienting means using a cue to look in the right place, then reorienting when the cue is misleading.

    Executive attention means following the rule when there is conflict, such as ignoring distracting arrows or shapes.

    Exploratory engagement is an in-app practice metric built from timing and touch precision. It is not a clinical score.

    Support, prompting, and data terms

    Prompt level shows the strongest prompt used on that trial, from independent through to full support.

    Support code records the kind of help used, such as verbal, visual, model, reduced field, or physical support.

    Independence status records whether the learner was independent, prompted, or assisted.

    Omission means the learner did not respond within the question window.

    Commission means the learner made an incorrect response when the rule required a different action or no action.

    RT variability means how inconsistent reaction times were across the session. Higher variability can suggest less stable performance.

    Artifact logs are observational markers such as movement, speech, blinks, cap adjustments, or interrupted trials that may matter for later EEG data quality.

    V1.3.2 CPAT variants and research tools

    Signal Station is a sustained-attention variant. The learner taps only when the green signal appears.

    Ticket Hunt is a selective-spatial variant. The learner searches for the gold ticket among similar distractors.

    Platform Pointer is an orienting variant. The learner uses the cue, then taps the train even if the cue misleads.

    Traffic Lights is an executive-control variant. The learner waits through the red signal, then taps when the light turns green.

    V1.3.2 also adds EEG-ready markers, active-control scaffolding, artifact logging, and family-specific dashboards so the same game-like variants can support classroom use and later research.
